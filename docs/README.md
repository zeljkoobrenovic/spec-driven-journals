# Generated Site Output

`docs/` is the publication output for this repository.

Journal directories such as `docs/principles/` and `docs/tech-strategy/` are
generated by:

```bash
python3 _wiring/build.py
```

The start page under `docs/start/` is generated by:

```bash
python3 _start/generate-docs.py
```

`docs/index.html` is a small redirect to `start/index.html`.

Do not hand-edit generated journal pages or start-page files. Edit the source
under `_journals/`, `_templates/`, or `_start/`, then rebuild.

Preview locally with:

```bash
python3 -m http.server 8000 --directory docs
```
