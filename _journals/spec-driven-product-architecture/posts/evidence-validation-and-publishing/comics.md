How an AI-authored product model earns trust — in eight panels.

![Comic panel: a robot unveils a polished model on a pedestal while an architect inspects it with a magnifying glass.](assets/images/evidence-validation-and-publishing/comic-01-polished-but-untrusted.jpeg)
**Panel 1:** *An AI agent can produce a polished model fast — but polish is not trust. The first job of the model is to be reviewable.*

![Comic panel: an architect sorts identical cards into trays labeled fact, assumption, inference, and question while a robot watches.](assets/images/evidence-validation-and-publishing/comic-02-fact-or-guess.jpeg)
**Panel 2:** *Product architecture mixes public facts, internal knowledge, and judgment. Review is only possible when the model says which is which.*

![Comic panel: a robot paints a fix directly onto a rendered documentation page while a generator machine prepares to overwrite it.](assets/images/evidence-validation-and-publishing/comic-03-fixing-the-page.jpeg)
**Panel 3:** *The anti-pattern: fixing the rendered page. It makes one page look better and breaks the model every future session depends on.*

![Comic panel: an architect connects a cable labeled grounding between a cloud-shaped concept board and a grid of reality signals.](assets/images/evidence-validation-and-publishing/comic-04-the-grounding-join.jpeg)
**Panel 4:** *The core move: grounding joins dreaming and exploring. Every important concept connects to evidence or is marked as an assumption.*

![Comic panel: a product brick connected by strings to small evidence cards with repository, cloud, and cost icons.](assets/images/evidence-validation-and-publishing/comic-05-evidence-cards.jpeg)
**Panel 5:** *A grounded brick points to repositories, services, and cost signals. The cards don't decide the architecture — they make it challengeable.*

![Comic panel: a robot feeds JSON pages through a validator gate that stamps checks and rejects one page flagged duplicate ID.](assets/images/evidence-validation-and-publishing/comic-06-validator-floor.jpeg)
**Panel 6:** *Validators prove the source parses, IDs are unique, and references hold. Mechanical validation is the floor — not strategic judgment.*

![Comic panel: an architect and colleagues debate a model at a review table while the robot serves coffee on the side.](assets/images/evidence-validation-and-publishing/comic-07-humans-still-judge.jpeg)
**Panel 7:** *The cost: discipline doesn't replace review. Domain boundaries, KPIs, ownership, and roadmap realism still need human judgment.*

![Comic panel: an architect and a robot stack labeled session blocks into a growing, stable tower.](assets/images/evidence-validation-and-publishing/comic-08-structured-memory.jpeg)
**Panel 8:** *The durable payoff: not a prettier site, but a living model — structured memory that humans and AI agents keep improving together.*
