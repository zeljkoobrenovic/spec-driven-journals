Durable writing deserves a home that humans and AI agents can both inspect — in eight panels.

![Comic panel: a robot carries a toppling pile of notes, chats, and documents while an engineer asks where a decision went.](assets/images/what-are-spec-driven-journals/comic-01-the-hook.jpeg)
**Panel 1:** *Meetings and chats are fine for creation — and terrible for memory.*

![Comic panel: an engineer and a robot look into a foggy archive of scattered tools while a small figure searches with a flashlight.](assets/images/what-are-spec-driven-journals/comic-02-the-problem.jpeg)
**Panel 2:** *Scattered across tools, records cannot be inspected, linked, or built on.*

![Comic panel: an engineer presents a tidy shelf of markdown booklets over a git symbol while a robot applauds.](assets/images/what-are-spec-driven-journals/comic-03-the-core-idea.jpeg)
**Panel 3:** *The core idea: treat durable writing as structured source with a stable home.*

![Comic panel: a robot ready to write a huge blank page while an engineer holds up a small one-page spec sheet.](assets/images/what-are-spec-driven-journals/comic-04-the-spec-contract.jpeg)
**Panel 4:** *The spec is the working contract — short on purpose, written before the article.*

![Comic panel: a conveyor belt turns markdown pages and a spec into framed web pages via a small build machine.](assets/images/what-are-spec-driven-journals/comic-05-three-layers.jpeg)
**Panel 5:** *Three layers: source is canonical, the build is small, the publication is output.*

![Comic panel: a robot tries to wrench a finished web page while an engineer redirects him to the markdown source.](assets/images/what-are-spec-driven-journals/comic-06-fix-the-source.jpeg)
**Panel 6:** *Generated pages are disposable — if output looks wrong, fix the source.*

![Comic panel: a robot presents a giant tangled machine while an engineer holds a tiny toolbox with one script.](assets/images/what-are-spec-driven-journals/comic-07-small-on-purpose.jpeg)
**Panel 7:** *Lightweight by design: standard library, templates, and nothing to install.*

![Comic panel: an engineer and a robot admire a wall of linked published pages, the robot holding a small spec sheet.](assets/images/what-are-spec-driven-journals/comic-08-the-closer.jpeg)
**Panel 8:** *The loop: source, spec, build, review — and fixes always flow back to source.*
