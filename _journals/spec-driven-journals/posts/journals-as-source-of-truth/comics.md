Why you edit the markdown and never the generated page — in eight panels.

![Comic panel: an engineer pins a card reading SOURCE WINS above markdown files and printed web pages while a robot salutes.](assets/images/journals-as-source-of-truth/comic-01-source-wins.jpeg)
**Panel 1:** *The most important rule in Spec-Driven Journals is simple: source wins.*

![Comic panel: a robot tries to stick a patch onto a rendered web page labeled docs while the engineer stops him.](assets/images/journals-as-source-of-truth/comic-02-the-tempting-fix.jpeg)
**Panel 2:** *The generated page is right there. Fixing it there is the trap.*

![Comic panel: a build machine replaces all generated pages while the robot's manual patch falls into a wastebasket.](assets/images/journals-as-source-of-truth/comic-03-the-build-overwrites.jpeg)
**Panel 3:** *During a build, docs is removed and recreated. Manual changes there do not survive.*

![Comic panel: a two-shelf diagram separating editable source files from generated pages marked read only.](assets/images/journals-as-source-of-truth/comic-04-two-layers.jpeg)
**Panel 4:** *The canonical layer is for editing; the generated layer is for reading and review.*

![Comic panel: engineer and robot walk a circular workflow loop from source to build to inspection and back to source.](assets/images/journals-as-source-of-truth/comic-05-the-feedback-loop.jpeg)
**Panel 5:** *Change source, build, inspect, return to source. The rendered page is feedback, not a canvas.*

![Comic panel: a robot confidently reads an organized file tree on one side, while a confused ghost robot guesses among scattered notes on the other.](assets/images/journals-as-source-of-truth/comic-06-agents-need-stable-context.jpeg)
**Panel 6:** *Agents inspect source before acting. Scattered truth means guessing, and guessing breeds confident wrong output.*

![Comic panel: the engineer edits a spec document first while the article and modality docs wait in line behind it.](assets/images/journals-as-source-of-truth/comic-07-spec-changes-first.jpeg)
**Panel 7:** *When a post changes direction, the spec changes first - that is how intent stays visible across sessions.*

![Comic panel: a new author opens a tidy source-tree drawer with labeled compartments while the engineer and robot look on.](assets/images/journals-as-source-of-truth/comic-08-the-practical-test.jpeg)
**Panel 8:** *The practical test: if the answers live in source, the journal keeps improving. If they live in memory, it decays.*
