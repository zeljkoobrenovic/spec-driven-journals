A journal is a small source tree you can read like a map — in eight panels.

![Comic panel: an engineer shows a small tree with folder-labeled branches to a curious robot holding a magnifying glass.](assets/images/anatomy-of-a-journal/comic-01-the-source-tree.jpeg)
**Panel 1:** *One config file, a posts folder, optional assets — that is the whole anatomy.*

![Comic panel: a robot hesitates between two doors, one bearing a glowing config.yaml badge and one plain.](assets/images/anatomy-of-a-journal/comic-02-folder-or-journal.jpeg)
**Panel 2:** *Two directories can look the same — only one is a journal.*

![Comic panel: an engineer flips a document-shaped wall switch and a table-of-contents receipt prints out for the robot.](assets/images/anatomy-of-a-journal/comic-03-config-is-the-switch.jpeg)
**Panel 3:** *config.yaml turns a folder into a journal and fixes the reading order.*

![Comic panel: a robot wheels a crate holding an article, a spec, and an assets box while an engineer nods.](assets/images/anatomy-of-a-journal/comic-04-everything-travels-together.jpeg)
**Panel 4:** *The per-post folder: article, spec, and media travel as one unit.*

![Comic panel: an engineer repaints a sign while stopping a robot from unbolting the address plaque locked under glass.](assets/images/anatomy-of-a-journal/comic-05-permalink-is-a-promise.jpeg)
**Panel 5:** *Front matter drives the page — and the permalink is a promise.*

![Comic panel: an engineer pins a blueprint beside a framed article on a corkboard while a robot reads its sections.](assets/images/anatomy-of-a-journal/comic-06-spec-beside-the-article.jpeg)
**Panel 6:** *spec.md is not polish — it is the contract published beside the post.*

![Comic panel: a robot files three documents into a slot while a browser window above grows matching tabs.](assets/images/anatomy-of-a-journal/comic-07-tabs-by-file-presence.jpeg)
**Panel 7:** *Summary, dialog, comic: discovered by file presence, no config changes.*

![Comic panel: a robot tries to edit a freshly printed page at a machine labeled docs while an engineer points to the source crate.](assets/images/anatomy-of-a-journal/comic-08-source-not-output.jpeg)
**Panel 8:** *docs/ is rebuilt every time — the journal lives in its source tree.*
