How a human and an AI agent run a journal together without the agent going rogue — in eight panels.

![Comic panel: an over-eager robot proudly prints a tall stack of articles while an engineer holds one page skeptically.](assets/images/ai-mediated-journal-operations/comic-01-not-autopilot.jpeg)
**Panel 1:** *AI-mediated does not mean AI-owned: the human states intent, the agent edits, the human reviews.*

![Comic panel: an engineer holds a polished document that fails to fit into a wall of organized journal folders like a wrong puzzle piece.](assets/images/ai-mediated-journal-operations/comic-02-plausible-prose.jpeg)
**Panel 2:** *The failure mode: fluent, plausible prose that does not fit Spec-Driven Journals.*

![Comic panel: a robot reads a book labeled AGENTS.MD in a library corner while an engineer nods approvingly.](assets/images/ai-mediated-journal-operations/comic-03-read-before-editing.jpeg)
**Panel 3:** *Read before editing: the repo's own files are cheaper than repairing a wrong guess.*

![Comic panel: an engineer and a robot hold a large page titled SPEC between them like a signed contract.](assets/images/ai-mediated-journal-operations/comic-04-spec-as-contract.jpeg)
**Panel 4:** *For non-trivial posts, the spec comes first: intent, audience, success criteria, non-goals.*

![Comic panel: an engineer redirects a robot away from painting over a framed webpage toward editing a plain source file.](assets/images/ai-mediated-journal-operations/comic-05-source-not-output.jpeg)
**Panel 5:** *Generated output is disposable. If a page is wrong, the fix lives in source.*

![Comic panel: a robot tiptoes through scattered folders and sticky notes, carrying only its own small folder overhead.](assets/images/ai-mediated-journal-operations/comic-06-dirty-trees.jpeg)
**Panel 6:** *Worktrees are shared and often dirty: build scoped, and never revert unrelated changes.*

![Comic panel: a robot runs a machine with four output trays labeled ARTICLE, SUMMARY, DIALOG, and COMIC while an engineer checks them.](assets/images/ai-mediated-journal-operations/comic-07-modality-skills.jpeg)
**Panel 7:** *Four skills, one read-only contract: every modality traces back to the same spec.*

![Comic panel: a robot hands an engineer a single tidy checklist page while a build terminal shows a green checkmark.](assets/images/ai-mediated-journal-operations/comic-08-the-report.jpeg)
**Panel 8:** *The payoff: enough structure that the agent does real work — and one report tells you what to review.*
